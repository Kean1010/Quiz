Solution for Quiz Party

The Problem

Your Quiz Party app uses browser's LocalStorage to store quiz data. When you host the app on GitHub Pages, participants cannot join because LocalStorage is not shared between different devices or browsers.

LocalStorage is limited to a single browser on a single device.

Quick Solutions

Solution 1: Test on Same Device

For testing purposes, you can create and join quizzes on the same device and browser:

  1. Open your GitHub Pages website in a browser
  2. Create a new quiz
  3. Copy the game code
  4. Open a new tab in the same browser
  5. Join the quiz using the copied code

This works because both tabs share the same origin and can access the same LocalStorage data.

Solution 2: Use URL Parameters

For a simple solution that works across devices, you can modify the code to use URL parameters:

  1. When creating a quiz, encode the quiz data in the URL
  2. Share this special URL with participants
  3. Participants can join directly through this URL

Implementation Guide

URL Parameter Solution

Here's how to modify your code to use URL parameters: 

// 1. Add this function to generate a shareable URL
function generateShareableUrl(quizData) {
    // Create a compact version of the quiz data
    const compactData = {
        t: quizData.title,
        d: quizData.description,
        q: quizData.questions.map(q => ({
            t: q.text,
            l: q.timeLimit,
            o: q.options,
            c: q.correctAnswerIndex
        }))
    };
    
    // Convert to JSON and encode
    const encodedData = btoa(JSON.stringify(compactData));
    
    // Create URL with encoded data
    return `${window.location.origin}${window.location.pathname}?quiz=${encodedData}`;
}
// 2. Modify the createQuizBtn click handler
createQuizBtn.addEventListener('click', () => {
    // ... (existing validation code)
    
    // Generate game code (still useful for display)
    const gameCode = generateGameCode();
    
    // Store quiz data in localStorage (for host)
    localStorage.setItem(`quiz_${gameCode}`, JSON.stringify(quizData));
    
    // Show success screen
    gameCodeElement.textContent = gameCode;
    
    // Generate shareable URL
    const shareableUrl = generateShareableUrl(quizData);
    
    // Generate QR code with the shareable URL
    new QRCode(qrCodeElement, {
        text: shareableUrl,
        width: 150,
        height: 150,
        colorDark: "#4361ee",
        colorLight: "#ffffff",
        correctLevel: QRCode.CorrectLevel.H
    });
    
    // Add a "Copy Link" button
    const copyLinkBtn = document.createElement('button');
    copyLinkBtn.className = 'mt-2 text-primary hover:text-primary/80 transition-all-300 flex items-center mx-auto';
    copyLinkBtn.innerHTML = '<i class="fa fa-link mr-1"></i> Copy Link';
    copyLinkBtn.addEventListener('click', () => {
        navigator.clipboard.writeText(shareableUrl).then(() => {
            copyLinkBtn.innerHTML = '<i class="fa fa-check mr-1"></i> Copied!';
            setTimeout(() => {
                copyLinkBtn.innerHTML = '<i class="fa fa-link mr-1"></i> Copy Link';
            }, 2000);
        });
    });
    
    // Add the button to the page
    const qrCodeContainer = document.querySelector('.qr-code-container');
    qrCodeContainer.appendChild(copyLinkBtn);
    
    // Show success screen
    quizFormContainer.classList.add('hidden');
    quizCreatedSuccess.classList.remove('hidden');
    
    // Store host information
    localStorage.setItem('isHost', 'true');
    localStorage.setItem('currentGameCode', gameCode);
});
// 3. Add this code to check for quiz data in URL on page load
window.addEventListener('DOMContentLoaded', () => {
    const urlParams = new URLSearchParams(window.location.search);
    
    // Check for quiz data in URL
    const quizData = urlParams.get('quiz');
    if (quizData) {
        try {
            // Decode and parse the quiz data
            const decodedData = JSON.parse(atob(quizData));
            
            // Reconstruct the full quiz data format
            const fullQuizData = {
                title: decodedData.t,
                description: decodedData.d,
                questions: decodedData.q.map(q => ({
                    text: q.t,
                    timeLimit: q.l,
                    options: q.o,
                    correctAnswerIndex: q.c
                }))
            };
            
            // Generate a game code
            const gameCode = generateGameCode();
            
            // Store the quiz data
            localStorage.setItem(`quiz_${gameCode}`, JSON.stringify(fullQuizData));
            
            // Redirect to join page with the game code
            window.location.href = `${window.location.origin}${window.location.pathname}?join=${gameCode}`;
        } catch (error) {
            console.error('Error parsing quiz data from URL:', error);
        }
    }
    
    // Existing code for checking join code
    const joinCode = urlParams.get('join');
    if (joinCode) {
        document.getElementById('join-game-code').value = joinCode;
        document.getElementById('join').scrollIntoView({ behavior: 'smooth' });
    }
});

Limitations

The URL parameter solution has some limitations:

  • URLs have a maximum length (usually around 2000 characters)
  • Very long quizzes might not fit in a URL
  • Participants must use the exact link to join

For a production app, you would need a backend service to store quiz data.
